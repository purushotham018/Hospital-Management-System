# Hospital Management System

A web-based Hospital Management System developed using Django to streamline hospital operations, including patient records, doctor management, appointments, and user authentication.

## Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- User Authentication
- CRUD Operations
- Responsive User Interface

## Tech Stack
- Python
- Django
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- Git & GitHub

## Installation

```bash
git clone https://github.com/purushotham018/Hospital-Management-System.git
cd Hospital-Management-System
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
